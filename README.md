# flask-backend-api
Backend RESTful API with Flask and SQLAlchemy

## Installation
```bash
$ virtualenv venv -p python3
$ source venv\bin\activate
$ pip install -r requirements.txt
$ python app.py
```

### URL
http://127.0.0.1:5000/api/v1
